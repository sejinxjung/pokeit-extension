# POKEIT - Visual QA Inspector

A Chrome extension that inspects DOM trees, styles (spacing, font, layout), and React component hierarchies in real time.

웹페이지의 DOM 트리, 스타일(간격, 폰트, 레이아웃), React 컴포넌트 계층을 실시간으로 검사하는 Chrome 확장 프로그램입니다.

**Website / 웹사이트:** [https://pokeit.app](https://pokeit.app)

## Installation / 설치 방법

### 1. Download / 다운로드
Download [pokeit-extension.zip](./pokeit-extension.zip).

[pokeit-extension.zip](./pokeit-extension.zip)을 다운로드합니다.

### 2. Unzip / 압축 해제
Extract the downloaded zip file.

다운로드한 zip 파일의 압축을 풀어주세요.

### 3. Load in Chrome / Chrome에 로드
1. Go to `chrome://extensions` in your address bar / Chrome 주소창에 `chrome://extensions` 입력
2. Enable **Developer mode** (top right) / 우측 상단 **개발자 모드** 활성화
3. Click **Load unpacked** / **압축해제된 확장 프로그램을 로드합니다** 클릭
4. Select the extracted folder / 압축 해제한 폴더 선택

### 4. Usage / 사용
- Click the POKEIT icon on any webpage / 아무 웹페이지에서 POKEIT 아이콘 클릭
- Click elements to inspect DOM tree, styles, and React component info / 요소를 클릭하면 DOM 트리, 스타일, React 컴포넌트 정보 확인 가능
- Add QA notes, then copy or delete per site from the popup / QA 노트 추가 후 팝업에서 사이트별 복사/삭제 가능
