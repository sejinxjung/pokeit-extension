# POKEIT - Visual QA Inspector

웹페이지의 DOM 트리, 스타일(간격, 폰트, 레이아웃), React 컴포넌트 계층을 실시간으로 검사하는 Chrome 확장 프로그램입니다.

## 설치 방법

### 1. 다운로드
[pokeit-extension.zip](./pokeit-extension.zip)을 다운로드합니다.

### 2. 압축 해제
다운로드한 zip 파일의 압축을 풀어주세요.

### 3. Chrome에 로드
1. Chrome 주소창에 `chrome://extensions` 입력
2. 우측 상단 **개발자 모드** 활성화
3. **압축해제된 확장 프로그램을 로드합니다** 클릭
4. 압축 해제한 폴더 선택

### 4. 사용
- 아무 웹페이지에서 POKEIT 아이콘 클릭
- 요소를 클릭하면 DOM 트리, 스타일, React 컴포넌트 정보 확인 가능
- QA 노트 추가 후 팝업에서 사이트별 복사/삭제 가능

## Chrome 웹 스토어
현재 웹 스토어 심사 중입니다. 승인되면 링크가 업데이트됩니다.
